#### [使用说明](usage.md)
1. 基于`Github`， 建立应用程序`repository`的备份
2. 在本地计算机安装, `node.js`， 建立应用程序`repository`的备份
3. 在应用程序中，配置知识库
4. 执行应用程序的部署脚本， 将应用程序部署到知识库中
5. 组织知识库，编写`markdown`文件
